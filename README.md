## Rodando o projeto

Clone the project and access the folder.

```bash
$ git clone git@github.com:vitorhugomarcal/coffee-delivey.git
$ cd coffee-delivey 
```

Follow the steps below:

```bash
# Install the dependencies
$ yarn install
# Start the project
$ yarn start
```

The app will be available for access on your browser
